About Paintcolor | A custom color palette is a collection of colors or color styles that can be saved and used in a design. Custom color palettes can help organize and store colors.

How to create a custom color palette

Choose individual colors or color styles Use the colors in a selected object or an entire document Use the Color Palette Manager in CorelDRAW to access available color palettes and create custom color palettes

Uses of custom color palettes

To customize the aesthetics of a site To make a block stand out against traditional color schemes To store all colors or color styles in a document

Color palette vs. color scheme

A color palette is a set of colors that can be used in a design. A color scheme is a specific combination of colors that are used together to create a certain aesthetic or to achieve a specific goal.
